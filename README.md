# SortVerse Infinity Match - 时空邮局

一款结合了自由物理交互、多维逻辑解谜与剧情合成机制的创新型Web端三消分拣游戏。

## 游戏特色

### 🌟 创新玩法
- **自由物理交互**：非传统网格的自由物理掉落与堆叠，物品可以自由移动、旋转和堆叠
- **多维消除逻辑**：不仅支持视觉相同，还支持逻辑关联、碎片拼接等复合消除条件
- **流畅触控交互**：支持鼠标拖拽、双击旋转、右键放置引力点

### 🎮 多种游戏模式

#### 1. 记忆碎片模式（推荐）
玩家扮演一位时空邮局的分拣员，负责整理来自不同时代、不同人物寄出的记忆包裹。每个包裹是一个模糊的片段，需要将关联的碎片拼合成完整的记忆。

**特色：**
- 消除条件变为逻辑匹配而非视觉相同
- 考验观察和推理能力
- 每个完整记忆被拼出后，播放一段感人的故事
- 难度升级：从2个碎片拼1件物品，到5个碎片拼一个场景

#### 2. 经典三消模式
传统的三消玩法，三个相同物品即可消除。

#### 3. 逻辑关联模式
找出逻辑上相关的物品组（如水果类、动物类、天气类等）。

### 📜 剧情故事
游戏包含10个感人的记忆故事：
- 祖父的怀表 - 1945年战场上的希望
- 未曾寄出的信 - 1928年的爱情祭奠
- 小美的布偶 - 1985年的母爱温度
- 无名英雄 - 抗美援朝的牺牲
- 摄影师的最后一卷 - 老北京的定格
- 迟到五十年的婚礼 - 养老院的重逢
- 十七岁的心事 - 梦想的实现
- 妈妈的摇篮曲 - 灯塔守望者
- 开往春天的列车 - 2008年的创业梦
- 守望者 - 孤岛四十年的坚守

### 🎨 视觉风格
- 泛黄纸张、老式邮戳、羊皮纸背景
- 蒸汽朋克元素
- 复古优雅的中文字体

## 技术架构

### 核心模块
```
├── index.html          # 主页面
├── styles.css          # 样式文件
├── config.js           # 游戏配置（关卡、记忆数据）
├── utils.js            # 工具函数
├── physics-engine.js   # 物理引擎封装（基于Matter.js）
├── match-system.js     # 匹配判定系统
├── item-generator.js   # 物品生成器
├── story-system.js     # 故事系统
├── ui-manager.js       # UI管理器
├── game-manager.js     # 游戏管理器
└── app.js              # 入口文件
```

### 核心特性实现

#### 物理引擎
- 使用 Matter.js 实现真实的物理模拟
- 支持重力、碰撞、摩擦、弹力
- 可放置引力点影响物品流动

#### 匹配系统
```javascript
// 三种匹配类型
matchType: 'visual'   // 视觉相同（经典三消）
matchType: 'logical'  // 逻辑关联（记忆碎片）
matchType: 'logic'    // 逻辑分组（分类匹配）
```

#### 交互系统
- **拖拽**：鼠标左键拖拽物品
- **选择**：单击物品选择/取消选择
- **旋转**：双击物品旋转90度
- **引力点**：右键或点击空白区域放置

## 快速开始

### 运行方式

#### 方式一：直接打开
用浏览器直接打开 `index.html` 文件即可游玩。

#### 方式二：本地服务器（推荐）
```bash
# 使用 Python
python3 -m http.server 8244

# 或者使用 Node.js (http-server)
npx http-server -p 8244

# 或者使用任何静态服务器
```

然后在浏览器中访问：`http://localhost:8244`

### 操作说明

| 操作 | 方式 |
|------|------|
| 拖拽物品 | 鼠标左键按住拖动 |
| 选择物品 | 单击物品 |
| 旋转物品 | 双击物品 |
| 放置引力点 | 右键或点击空白区域 |
| 确认匹配 | 选择3个物品后点击"确认匹配" |

## 游戏规则

### 记忆碎片模式
1. 物品从上方掉落，带有真实的物理效果
2. 单击选择属于同一个记忆的3个碎片
3. 选择完成后点击"确认匹配"
4. 匹配成功后播放该记忆的故事
5. 完成目标数量的记忆后进入下一关

### 关卡信息
| 关卡 | 名称 | 目标记忆 | 所需碎片 | 时间限制 |
|------|------|----------|----------|----------|
| 1 | 初识记忆 | 3个 | 3个 | 无 |
| 2 | 时光印记 | 5个 | 3个 | 无 |
| 3 | 岁月珍藏 | 7个 | 4个 | 3分钟 |
| 4 | 记忆长河 | 10个 | 5个 | 4分钟 |

## 开发说明

### 添加新的记忆故事
在 `config.js` 中的 `memorySets` 对象添加新条目：

```javascript
memorySets: {
    my_new_memory: {
        id: 'my_new_memory',
        name: '我的记忆名称',
        category: '分类',
        era: '年代',
        fragments: [
            { id: 'frag1', name: '碎片1', icon: '📷', color: '#颜色', description: '描述' },
            { id: 'frag2', name: '碎片2', icon: '📝', color: '#颜色', description: '描述' },
            { id: 'frag3', name: '碎片3', icon: '🎁', color: '#颜色', description: '描述' }
        ],
        story: {
            title: '故事标题',
            text: '完整的故事内容...'
        }
    }
}
```

### 添加新关卡
在 `config.js` 中的 `levels` 数组添加：

```javascript
{
    id: 5,
    name: '新关卡名称',
    targetMemories: 8,
    requiredFragments: 4,
    timeLimit: 180,
    memorySets: ['memory_set_id1', 'memory_set_id2', ...]
}
```
